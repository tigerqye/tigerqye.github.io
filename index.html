EXPERIENCE

VirtaMove Software Developer

May 2021 - Aug 2021

This was my very first coop! VirtaMove is a company that creates software to migrate server applications to new cloud or datacenter servers. Their main 2 products are V-migrate, a 1 to 1 migration application and VirtaMaestro an online multi-migration application.

One of new things I learned was Robot Framework an automation framework. I programmed Robot Framework scripts for the IIS migration on V-migrate and the migration of IIS through Maestro. Additionally I was able implement alot of new features on their Maestro product. I wrote C# code to code to export backend data to csv files and pdf reports for use by users. Additionally I created the entirely new Help Center menu for user help. This included implementing links to documentation, support tickets and a page inspector tool which would show the user all the important page components and what they do.

FRC - First Robotics Competition

Sep 2018 - Jun 2020

FRC has been my gateway into a variety of hands on skills such as using drill presses, band saws, milling machines and much, much more. FRC taught me how to apply the theoretical knowledge I learned in classrooms to part design and manufacturing.

My role in team 1246 (our FRC team) was overall/design lead and thus I was responsible for integrating other designers CAD models into the final design and providing tasks for members of the team. Some of my design tasks included designing the elevator, powercell (balls) storage system and the hooking mechanism for the climb.

Probably one of my proudest achievements was using Fusion 360 to design a mechanical single use hook for 2020 FRC robot. A single use hook was needed to ensure the elevator would not descend during the power off period at the end of the game. The key component of the mechanism is the spring that was used such that no electrical components were required to latch on to the elevator.

Midnight Sun - UW Solar Car Team

Feb 2021 - Present

Midnight Sun was the natural continuation after FRC. At Midnight Sun I am mostly involved with designing SOLIDWORKS components such as mounting hardware and snap covers for the battery box, the door handle mechanism for locking and opening the solar car and adding other miscellaneous hardware to the assembly.

Cover Letter Template Generator

Least Torque Calculator

Wrote a C++ script to determine the 3 link lengths that produce the least torque of a three-degree-of-freedom (DOF) planar serial robotic manipulator arm given 3 final link positions and angle w.r.t the x-axis. The code iterated through all combinations of arm lengths from 1 to 1000 meters to find the combination that gave the least combined torque of the 3 positions. The code can be found here!
